    <h2 data-section-id="rzkdgm" data-start="330" data-end="341">Overview</h2>
    <p data-start="343" data-end="470">This project is an open-source Tarot
      application that I am developing as both a programming project and a
      spiritual study tool.</p>
    <p data-start="472" data-end="695">The application is designed to provide
      beautiful, informative, and meaningful Tarot readings while also serving
      as an educational resource for anyone interested in the symbolism,
      history, and interpretation of the 78 cards.</p>
    <p data-start="697" data-end="851">Although this project is intended to be
      enjoyable to use, it is also an exercise in software engineering, database
      design, and user interface development.</p>
